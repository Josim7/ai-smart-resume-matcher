# Senior AI Engineer Assignment – AI Smart Resume Matcher

## Overview
This project implements an AI-powered Resume Matching System that evaluates and ranks resumes based on a given job description using NLP techniques.

## Features
- Text preprocessing using NLP
- TF-IDF vectorization
- Cosine similarity scoring
- Resume ranking based on relevance

## Tech Stack
- Python
- Scikit-learn
- NLTK
- Pandas
- NumPy

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the project:
   python src/app.py

## Output
Ranked resumes with similarity scores.
